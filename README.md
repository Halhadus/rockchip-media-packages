# Rockchip Multimedia Build System (Debian Trixie)

This repository contains automated scripts and CI workflows to build Rockchip multimedia packages (MPP, RGA, FFmpeg with HW accel) and **Firefox** with hardware decoding support for **Debian Trixie (ARM64)**.

Designed for Orange Pi 5 series / Rock 5 series and similar RK3588 based devices running Debian Trixie.

## üì¶ Included Packages

The build system compiles the following components:
- **v4l-utils** (Patched for libv4l-rkmpp)
- **mpp** (Rockchip Media Process Platform)
- **linux-rga** (2D Raster Graphic Acceleration)
- **ffmpeg** (Patched with MPP & RGA support)
- **libv4l-rkmpp** (V4L plugin for Rockchip)
- **drm-cursor** (really I don't know what is that)
- **firefox-mpp** (Firefox 139.x patched with V4L2/MPP support for HW decoding) (thanks to hbiyik for patch)
- **pipewire** (Patched to enable **FDK-AAC** and **LDAC** for high-quality Bluetooth audio)
- **mpv** (Build against patched FFmpeg/MPP for hardware decoding support via rkmpp) (thanks to hbiyik for patch)

## üöÄ Building packages

You don't need to pollute your host system. Just use Docker.

*Note: Building Firefox takes a significantly long time (4-6 hours on ARM64).*

```bash
# Be sure Docker installed on your system
git clone https://github.com/Halhadus/rockchip-media-packages.git --single-branch --depth 1
cd rockchip-media-packages
chmod +x run.sh
./run.sh
```

## Debian Repository

Also you can use Debian repository from Github Pages

/etc/apt/preferences.d/media-packages:
```
Package: *
Pin: origin "halhadus.github.io"
Pin-Priority: 990
```

/etc/apt/sources.list.d/media-packages.sources:
```
Types: deb
URIs: https://halhadus.github.io/rockchip-media-packages/
Suites: trixie
Components: main
Trusted: yes
```
